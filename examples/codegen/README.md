# Codegen Example

This example demonstrates the `drel generate` workflow.

## Structure

```
codegen/
  drel.yaml           — drel config pointing to ./models
  models/
    task.go           — model using drel.Model[int] with unexported fields
  models/
    task_drel.go      — GENERATED by drel generate
  db/
    drel_gen.go       — GENERATED aggregated DB struct
  main.go             — uses the generated code
```

## Steps

```bash
# 1. Generate code from model definitions
cd examples/codegen
go run ../../cmd/drel generate

# 2. Run the example (requires Postgres)
export DATABASE_URL="postgres://localhost:5432/drelexample?sslmode=disable"
go run .
```

## What `drel generate` produces

From a model like:

```go
type Task struct {
    drel.Model[int]
    title    string `db:"title"`
    done     bool   `db:"done"`
    priority int    `db:"priority"`
}
```

It generates:
- Column references (`var Tasks = struct { ... }`)
- Scan function (`scanTask`)
- Snapshot/diff functions for change tracking
- `TaskMeta` with all metadata wired up
- Aggregated `DB` struct with typed repository
